《IPFS 从入门到精通》
===================


在接触 IPFS 之后，对这种可以本质上改变网络数据的分发机制的协议非常感兴趣，但由于目前国内并没有比较深入分析 IPFS 的书籍，而一些资料因为英文的缘故在一定程度上也为不少 IPFS 爱好者带来了一些学习上的困扰，不仅为了加快扩散 IPFS 爱好者的国内群体，决定以开源的形式完成这本《IPFS 从入门到精通》的书籍。
